# aerogear-security-picketlink - Security Provider Implementation

## Getting started

        <dependency>
             <groupId>org.jboss.aerogear</groupId>
             <artifactId>aerogear-security-picketlink</artifactId>
             <version>1.1.0</version>
             <scope>compile</scope>
        </dependency>
